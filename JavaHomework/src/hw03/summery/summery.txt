1. java.lang 패키지 : 자바가 제공하는 가장 기본적인 클래스들리 포함되어 있는 패키지, import 없이 사용 가능
	1.1 Object 클래스 : 모든 클래스가 상속받는 최상위 클래스
	1.2 String 클래스 : 문자열을 사용하는 클래스
		함수 / 반환형 / 설명
		charAt(int index), char, index 위치에 있는 문자 반환
		...
	1.3 StringBuffer 클래스와 StringBuilder 클래스
		- String 클래스와 다르게 문자열 변경시 새로운 객체를 생성하지 않고, 버퍼에 담겨있는 문자열을 바로 수정
		- StringBuffer 클래스와 StringBuilder 클래스의 기본 동작은 같지만, StringBuffer 클래스는 동기화를 지원한다.
	1.4 Math 클래스
		- 숫자 계산에 사용되는 기능을 가진 클래스
		- 모든 메소드들이 정적 메소드로 정의되어 있어 객체를 생성하지 않고 사용한다.
		메소드 / 설명
		
2. Wrapper 클래스 : 기본형 데이터타입을 클래스화하기 위해 사용하는 클래스, 클래스화 하기 때문에 다형성을 활용해 Object 클래스를 이용할 수 있다.
	기본형 데이터타입 / Wrapper 클래스
	2.1 박싱(Boxing), 언박싱(Unboxing)
		2.1.1 박싱 : 기본형 데이터타입을 객체화 시키는 작업
			int i = 10;
			Integer iVal1 = new Integer(i);
			Integer iVal2 = new Integer(20);
		2.1.2 언박싱 : 객체에 저장되어 있는 데이터를 기본형 데이터타입으로 꺼내는 작업
			Integer iVal = new Integer(50);
			int i = iVal.intValue();
	2.2 자동 박싱, 자동 언박싱 : JDK1.5 이상부터 지원
